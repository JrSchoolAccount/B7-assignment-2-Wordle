About High-Score
Wordle-Clone project

Fullstack project with frontend, backend and database support.

Frontend:

Frontend is developed with React using jsx, styled with Tailwind and bundled with Vite.

Backend:

The backend run on nodejs using express with ejs view engine for SSR and mongoose to interact with the database.

Database:

The database used for this project is mongoDB, both database on localhost and on mongoDB Atlas are supported.
